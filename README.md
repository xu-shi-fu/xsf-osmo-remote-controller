# XORC

XORC (XSF-OSMO-Remote-Controller, 徐师傅的 OSMO 遥控器) , 是一个 Android App, 用于遥控 DJI OSMO 手持摄影设备.
它提供了一个无头 (headless) 的快速遥控方式, 以应对复杂且要求高效的使用场景.

## 获取源代码

通过以下命令, 把项目 Clone 到本地:

    git clone https://github.com/xu-shi-fu/xsf-osmo-remote-controller.git
